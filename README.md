# もやもやメーター

**白黒つけない、意思決定ツール。** 二択・多択の迷いを、チェックボックスではなく **0.0〜1.0 のスライダー（Float）** でほぐす。

🔗 https://osakenpiro.github.io/moyamoya/

## これは何

「やる／やらない」「A／B」を二値（Boolean）で決めようとすると苦しい。本当の気持ちはたいてい **0.62** や **0.48** みたいな、あいだの値にある。

もやもやメーターは、

1. 迷っている **選択肢**（2〜6個）を並べ、
2. 大事にする **観点** に「重要度」スライダーをつけ、
3. 各選択肢が各観点にどれだけ当てはまるかを連続値で置き、
4. 加重平均で **選択肢ごとの Float スコア（0〜1）** を出す。

## 効きどころ＝近接判定

上位ふたつのスコア差で、判定の言い方が変わる。

| 差 | 判定 | 意味 |
|---|---|---|
| ≥ 0.15 | **白黒、ついた** | 迷わなくていい。決めて進む |
| 0.05〜0.15 | **ほぼ互角** | 最後は気持ちで決めていい範囲 |
| < 0.05 | **これは Float な問い** | もともと白黒つく問いじゃなかった。どちらでもいい／混ぜる／保留が正解 |

僅差を「決められない」ではなく「**Boolean で問うこと自体が間違いだった**」と読み替える——これが Boolean→Float の核。0.5を、許す。

## 仕様

- 単一HTMLファイル。依存ゼロ。サーバーレス（GitHub Pages）
- データはブラウザの localStorage だけに保存。サーバーには何も送らない
- 「共有」＝状態をURLのfragmentにbase64urlで埋める（メール・チャットでも壊れない）。受け取った人はリンクを開くだけで同じ盤面が復元される

## ローカルで動かす

```
python -m http.server 8000

```
